# 好感度评分系统

## 评分哲学

感情信号不是二元的，但人需要清晰的答案。这套评分系统的目标是：
**用客观可观察的行为信号，尽量排除主观投射，给出一个有参考价值的估算。**

评分不是命运，是信息。

---

## 七大维度详细标准

### 维度 1：主动性（权重 25%）

衡量"谁更想接触谁"。

| 信号 | 加分 |
|------|------|
| ta 经常第一个发消息 | +2 |
| ta 在没有明显理由时主动联系（只是想聊） | +2 |
| ta 主动约你出去 | +2 |
| ta 在你沉默时会"找话题"续聊 | +1 |
| ta 在忙碌时也抽空联系你 | +1 |
| 双方主动频率大致相当 | +1 |

| 扣分信号 | 扣分 |
|---------|------|
| 几乎总是你主动 | -3 |
| ta 只在有事/需要帮忙时联系你 | -2 |
| ta 对你的主动常常无视或冷处理 | -2 |

---

### 维度 2：回应质量（权重 20%）

衡量"ta 是否真的在乎这段对话"。

| 信号 | 加分 |
|------|------|
| 经常秒回或快速回复 | +1.5 |
| 回复内容有实质内容，不只是"嗯""哦""哈哈" | +2 |
| ta 的消息比你的长/更有内容 | +1 |
| 会接话/追问/延续话题 | +1.5 |
| 深夜还在回复你（且非紧急事项） | +2 |
| 即使很忙也会解释并补发 | +1 |

| 扣分信号 | 扣分 |
|---------|------|
| 经常已读不回 | -2 |
| 回复质量明显低于 ta 对其他人 | -2 |
| 单字/表情回复为主 | -1.5 |
| 明显在敷衍 | -2 |

---

### 维度 3：独特性（权重 20%）

衡量"你在 ta 眼里是否特别"。这是最重要的信号之一。

| 信号 | 加分 |
|------|------|
| ta 对你的态度和对其他朋友明显不同 | +3 |
| ta 给你起了昵称/只对你这样叫 | +2 |
| ta 只告诉你某些事（秘密分享） | +2 |
| ta 在你面前表现出在别人面前没有的一面 | +2 |
| ta 会特意关注你的动态 | +1 |

| 扣分信号 | 扣分 |
|---------|------|
| ta 跟你的相处方式和跟其他人完全一样 | -2 |
| ta 在你面前表现出明显的"保持距离"感 | -2 |
| ta 把你归入"普通朋友"那类明确表示 | -3 |

---

### 维度 4：情感暴露（权重 15%）

衡量"ta 是否愿意在你面前放下防备"。

| 信号 | 加分 |
|------|------|
| ta 会跟你分享烦恼、脆弱、内心挣扎 | +2 |
| ta 在你面前承认自己的失误或不好意思的事 | +1.5 |
| ta 跟你说过家人/感情方面的私事 | +2 |
| ta 会在你面前哭/崩溃 | +2 |
| ta 主动问你对自己的看法 | +1 |

| 扣分信号 | 扣分 |
|---------|------|
| 对话始终停留在表面/安全话题 | -1.5 |
| ta 明显在你面前刻意维持形象 | -1 |

---

### 维度 5：行为信号（权重 10%）

衡量"ta 的行动是否在说明问题"。

| 信号 | 加分 |
|------|------|
| ta 记得你说过的小细节 | +2 |
| ta 主动创造和你单独在一起的机会 | +2 |
| ta 给你带过东西/送过礼物 | +1.5 |
| ta 在你不舒服/需要帮助时主动出现 | +1.5 |
| 肢体接触增多（且不是 ta 对所有人都这样） | +2 |
| ta 会在你面前紧张/特别在意形象 | +1 |

| 扣分信号 | 扣分 |
|---------|------|
| 约好的事经常爽约或推脱 | -2 |
| 从不主动制造见面机会 | -1.5 |

---

### 维度 6：未来导向（权重 5%）

衡量"ta 是否在想象你们的未来"。

| 信号 | 加分 |
|------|------|
| ta 用"我们"描述你们的关系 | +2 |
| ta 说"下次我们去..."/"以后你来..." | +2 |
| ta 把你纳入自己的计划（跟你说"那时候你来不来"） | +2 |
| ta 对你的未来计划表现出真实的兴趣 | +1 |

| 扣分信号 | 扣分 |
|---------|------|
| 对话中从来没有涉及未来/下次 | -1 |

---

### 维度 7：直觉修正（权重 5%）

用户自己的感受是有效信息，但需要校准。

- 如果用户说"我自己感觉 ta 喜欢我"但列举不出具体理由：这可能是投射，轻微减分
- 如果用户说"我自己感觉 ta 喜欢我"并能列举具体细节：直觉有依据，正常加分
- 如果用户说"ta 应该不喜欢我"但原材料显示信号不错：询问用户是否有低自我评价倾向
- 如果用户说"ta 不喜欢我"且原材料支持这个结论：不要盲目安慰

---

## 综合评分计算

```
最终分 = Σ(各维度分 × 权重)
```

打分范围：各维度 0-10 分（可以有小数）

最终分四舍五入到小数点后一位。

---

## 特殊信号（一票权）

以下信号出现时，无论其他维度分数如何，需要特别说明：

### 强烈正向信号（加分 1-2 分）
- ta 主动问"你有没有喜欢的人"或类似话题
- ta 嫉妒心明显（ta 主动问你跟某人的关系）
- ta 主动对你坦白自己的感情状态
- 朋友说"ta 提过你"/"ta 好像喜欢你"

### 危险信号（减分 1-3 分，并单独提醒）
- ta 有明确的伴侣
- ta 明确说过"我们只是朋友"
- ta 跟你聊的时候同时在撩其他人
- ta 的行为模式是对所有人都这样（高亲和力 ≠ 有意思）
- ta 只在特定需求下找你（情绪垃圾桶、工具人）
